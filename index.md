### Java
```java
public class SumExample {
    public static void main(String[] args) {
        int[] numbers = {1, 2, 3, 4, 5};
        int total = 0;
        for (int number : numbers) {
            total += number;
        }
        System.out.println("The sum is: " + total);
    }
}
```

### C++
```cpp

    std::vector<int> numbers = {1, 2, 3, 4, 5};

    return 0;
}
```
